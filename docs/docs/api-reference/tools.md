# Tools Catalog

Complete list of all available tools.

## Tool Management

- `add_tool` - Add a new tool
- `get_tool` - Get tool by name
- `list_tools` - List all tools
- `update_tool` - Update a tool
- `remove_tool` - Remove a tool

## Agent System

- `generate_thoughts` - Generate thoughts from user query
- `generate_plan` - Generate plan from thought
- `execute_task` - Execute a task (plan)
- `resume_task` - Resume paused task

## Intelligence Features

- `get_similar_tasks` - Find similar past tasks
- `get_tool_recommendations` - Get tool recommendations
- `predict_plan_quality` - Predict plan success
- `track_cost` - Track token usage and costs

See [Tool Categories](../features/overview.md) for detailed documentation.
