# Introduction

Welcome to the **Agents MCP Server** documentation!

## What is Agents MCP Server?

Agents MCP Server is an intelligent Model Context Protocol (MCP) server that combines AI-powered agents with persistent memory, learning capabilities, and smart features. It serves as both an MCP server and client, enabling you to:

- **Manage Tools**: Store, search, and execute MCP tools from remote servers
- **Run AI Agents**: Execute complex tasks using AI agents (Thought, Planner, Executor)
- **Learn and Improve**: Memory system that learns from past executions
- **Benchmark Performance**: Track and optimize system performance
- **Make Smart Decisions**: Quality prediction, tool recommendations, and cost optimization
